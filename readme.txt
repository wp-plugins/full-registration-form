Extend your registration form with the user profile fields

== Description ==

With this plugin you add the fields 'First Name', 'Last Name', 'Nickname', 'Description', 'Website' to your registration form

== Installation ==

Aktivate the plugin.

== Frequently Asked Questions ==

=Where are the AIM / Yahoo / Jabber fields?=

Sorry no filter in WP 2.3.3 for these fields during the registration process
